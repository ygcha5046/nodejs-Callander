#Terminal Version
---
features, commands, examples

The calendar view, melo, today's date and the day you have the schedule are shown in color.

adding schedules, melo add \[Date] \[Content], melo add 2026-04-25 birthday party

Displays list view, melolist, and all saved schedules by date.

delete schedule, melodel \[date], melodel 2026-04-25 (delete date schedule)

Delete the entire initialization, melo del all, and all stored data.
